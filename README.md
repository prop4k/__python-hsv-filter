# Фильтр насыщенности изображений (HSV) 
## Функционал

* **Графический интерфейс** на базе `Tkinter` с вкладками "Оригинал" и "Результат".
* **Загрузка и сохранение** изображений в форматах JPG, PNG и др.
* **Интерактивный ползунок** для настройки минимального порога насыщенности (0-255).
* Конвертация изображений из BGR в HSV и обратно с помощью библиотеки **OpenCV**.
* Изолированная обработка канала насыщенности (S) с обнулением блеклых участков (эффект выделения ярких объектов на сером фоне).
